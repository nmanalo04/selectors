This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Inside my-div
class

Classes can be applied to more than item on a page.

Here's how we hook class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

compound

Compound selector allows more than one selector block to be seperated by a comma.

Here's how the CSS is applied to the elements:

 li.new-list,div#new-div{
    background-color:blue;
}

descendant

descendant selector let us access elements at any depth within the current element

Here's how the CSS is applied to the elements:

 li.new-list,div#new-div{
    background-color:red;
}

child

A child selector allows us to access the next immediate elements inside the current element

Here's how the CSS is applied to the elements:

 ul.kids>li{
    background-color:green;
}
